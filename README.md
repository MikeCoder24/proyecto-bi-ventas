# 📊 Análisis de Ventas – Proyecto BI

## 🚀 Objetivo

Analizar el desempeño de ventas para identificar productos más rentables, mercados principales y tendencias en el tiempo.

---

## 🛠️ Tecnologías utilizadas

* Python (Pandas)
* Power BI
* CSV

---

## 📂 Dataset

El dataset contiene información de ventas con las siguientes columnas:

* fecha
* producto
* categoria
* precio
* cantidad
* pais

---

## 📈 Análisis realizado

* Cálculo de ventas totales (precio × cantidad)
* Análisis de ventas por producto
* Análisis de ventas por país
* Análisis de ventas por mes

---

## 🔍 Insights principales

* 📌 Laptop es el producto con mayor generación de ingresos
* 📌 México es el país con mayor volumen de ventas
* 📌 Se observa un pico de ventas en el mes de abril

---
